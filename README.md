# 轻昼CE

继承Lua开发的轻昼Ultra，使用Kotlin+Jetpack Compose开发，采用Material You设计，拥有**更快的速度、更小的占用、美观的UI，且包含诸多实用功能以及社区功能**。

## 技术栈

- 语言：Kotlin
- UI 框架：Jetpack Compose
- 设计规范：Material You

## 功能特性

- ⚡ 更快的运行速度
- 📦 更小的安装包体积
- 🎨 美观的 Material You 界面
- 🔧 丰富的实用工具
- 👥 社区互动功能
